# powerbi-sales-dashboard
A Power BI project designed to analyze sales performance, visualize key metrics, and uncover business insights. The report includes interactive dashboards, trend analysis, and dynamic filtering to support data-driven decision-making
## Overview

This Power BI project provides a complete analysis of sales performance
using multiple business dimensions.\
It includes KPIs, visual insights, and interactive pages that help users
explore trends, customer behavior, product performance, and
profitability.

## Data Model

### Dimensions

-   Date Dimension
-   Customer Dimension
-   Product Dimension (Category & Subcategory)

### Fact Table

-   Sales Transactions

## Key Measures

-   Total Sales\
-   Total Orders\
-   Total Products Sold\
-   Total Customers\
-   Average Discount\
-   5 KPI Measures

## Report Pages (7 Pages)

1.  **Main Page** -- Full overview with KPIs\
2.  **Sales Page** -- Trends and breakdowns\
3.  **Profit Page** -- Profitability insights\
4.  **Hierarchy Page** -- Product hierarchy with drill-down\
5.  **Customer Insights Page** -- Customer segmentation & behavior\
6.  **Date Analysis Page** -- Time intelligence visuals\
7.  **Subcategory Page** -- Detailed subcategory performance

## Visuals Used

-   Donut charts\
-   Bar & Column charts\
-   Tree Map\
-   Decomposition Tree\
-   KPI Cards\
-   Line charts\
-   Drill-down hierarchy visuals
